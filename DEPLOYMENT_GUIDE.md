## Pre-Deployment Checklist

### 1. MongoDB Setup (IMPORTANT!)
Aapko MongoDB Atlas account banana hoga (free tier available):

1. **MongoDB Atlas Account banao:**
   - Visit: https://www.mongodb.com/cloud/atlas/register
   - Sign up for free account
   - Create a new cluster (M0 Free tier)

2. **Database User banao:**
   - Database Access ‚Üí Add New Database User
   - Username aur password set karo (yaad rakhna!)
   - Built-in Role: "Atlas admin" select karo

3. **Network Access setup:**
   - Network Access ‚Üí Add IP Address
   - "Allow Access from Anywhere" select karo (0.0.0.0/0)
   - Confirm karo

4. **Connection String copy karo:**
   - Clusters ‚Üí Connect ‚Üí Connect your application
   - Driver: Python, Version: 3.11 or later
   - Connection string copy karo, example:
   ```
   mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/depression_db?retryWrites=true&w=majority
   ```
   - `<password>` ko apne actual password se replace karo

## Common Issues & Solutions

### Issue 1: Models not loading
**Solution:** Models GitHub mein push ho gaye hain, automatically load honge

### Issue 2: MongoDB connection failed
**Solution:** 
- MONGO_URI check karo
- MongoDB Atlas mein IP whitelist check karo (0.0.0.0/0 hona chahiye)
- Username/password correct hai?

### Issue 3: App crashes on startup
**Solution:**
- Render logs check karo
- Environment variables sahi set hain?
- requirements.txt mein saari dependencies hain?

### Issue 4: Slow first load
**Solution:** Free tier pe app sleep mode mein jaata hai. First request slow hoga (cold start).
